# 🪨 📄 ✂️ Rock Paper Scissors

> A simple and interactive web-based version of the classic Rock, Paper, Scissors game.
> This project features instant color-coded feedback and a clean layout,
> allowing users to play against the computer smoothly using HTML, CSS, and JS.

---

### 🎮 Features

#### 👤 User Input Players can select:

```text
Rock 🪨 | Paper 📄 | Scissors ✂️
Input is handled through large, clickable emoji buttons.

🤖 Computer Choice
The system generates a random choice for the computer each round.

🏆 Result Display
Clearly shows the result after each round with dynamic background colors:
  Win  🟢 (Green)
  Loss 🔴 (Red)
  Tie  🔵 (Blue)

📊 Score Tracking Keeps track of:
Player Choice
Computer Choice
Match Outcome
Displays selections and round results instantly on the screen.

🎨 User-Friendly UI
Clean layout with hover effects
Large fonts and bright colors
Easy to understand and navigate

📱 Responsive Design

Works smoothly on desktop screens, tablet, and mobiles.
