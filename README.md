# 🪨 📄 ✂️ Rock Paper Scissors

> This is a clean, interactive, and web-based version of the classic Rock, Paper, Scissors game. 
> Built using simple web technologies, it allows users to play against a computer with instant visual feedback.

---

### 🎮 Features

#### 👤 User Input Players can select:

```text
Rock 🪨 | Paper 📄 | Scissors ✂️
Players select their move by clicking on the large, colorful emoji buttons.


🏆 Result Display
The game immediately shows the choices and changes the result background color:
  🟢 GREEN  = YOU WIN!
  🔴 RED    = YOU LOSE!
  🔵 BLUE   = IT'S A TIE!
